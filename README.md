# Hidrantes

Aplicacion para controlar hidrantes y armarios con flujo tipo Correctivos.

## Incluye

- Importar Excel y anadir solo registros nuevos.
- Filtrar por edificio, numero y tipo/otros.
- Editar registros y anadir nuevos.
- Tipo de equipo: Hidrante o Armario.
- Defectos diferentes segun el tipo.
- Campos Otro 1, Otro 2 y Otro 3.
- Dos fotos por registro, con opcion de eliminar/cambiar.
- Marcar registros como vistos o pendientes.
- Descargar Excel con datos, defectos y fotos.
- Generar Word de correctivos agrupado por cliente y defecto.

Los datos y fotos se guardan en el navegador/movil mediante IndexedDB.
